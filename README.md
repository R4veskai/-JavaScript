README - Лендинг для строительной компании

Описание проекта Этот проект представляет собой современный одностраничный лендинг для строительной компании, разработанный на React. Сайт демонстрирует услуги компании, реализованные проекты и предоставляет функционал для сбора заявок от потенциальных клиентов. Проект имеет модульную структуру, адаптивный дизайн и ориентирован на конверсию посетителей в заявки.

Запуск проекта Предварительные требования Node.js версии 14.x или выше npm (поставляется с Node.js) Браузер для просмотра

Установка и запуск Клонирование и переход в директорию проекта

git clone <URL-репозитория> cd <название-папки-проекта> Установка зависимостей

npm install Запуск в режиме разработки

npm start Приложение откроется автоматически в браузере

Функционал компонентов Header (Header.jsx) Логотип строительной компании Навигационное меню по разделам сайта Ссылки на социальные сети компании Адаптивное меню для мобильных устройств

Hero (Hero.jsx) Главный баннер с ключевым сообщением Призыв к действию (CTA) "Получить консультацию" Фоновое изображение строительного объекта Преимущества компании в виде текста

Services (Services.jsx) Карточки услуг с изображениями

Projects (Projects.jsx) Портфолио выполненных работ с фотографиями Галерея проектов с возможностью просмотра

Advantages (Advantages.jsx) Секция преимуществ компании ("О нас") Информация о компании, этапы работы

ContactForm (ContactForm.jsx) Форма обратной связи для заявок Валидация полей ввода (имя, телефон)

Footer (Footer.jsx) Контактная информация (адрес, email) Ссылки на социальные сети

Стили и дизайн CSS-модули: Каждый компонент имеет собственные стили Адаптивный дизайн: Полная поддержка мобильных устройств и планшетов Современные UI-элементы: Интерактивные кнопки, формы, карточки Цветовая схема: Корпоративные цвета строительной компании Типографика: Четкая иерархия заголовков и текста

Структура проекта в GitHub task.md - Цели и задачи проекта terms.md - Техническое задание (ТЗ) design/ - Юзер флоу и прототипы сайта review.md - Критерии оценки проекта optimize.md - Рекомендации по оптимизации сайта remark.md - Предложения по улучшению функционала grow.md - Git Flow prstrategy.md - PR-стратегия продвижения сайта и компании
