README - Лендинг для строительной компании
1. Описание проекта
Этот проект представляет собой современный одностраничный сайт, разработанный на React. Сайт демонстрирует услуги, проекты и предоставляет функционал для связи с компанией. Проект имеет модульную структуру и адаптивный дизайн.

2. Структура
app
 public
    foto #Изображения для сайта
 src
    components  #React-компоненты
         Advantages.css/jsx # Информаиця о нас
         ContactForm.css/jsx # Форма сбора заявок
         Footer.css/jsx # Футер
         Header.css/jsx # Хедер
         Hero.css/jsx # Главная страница
         Projects.css/jsx # Реализованные проекты
         Services.css/jsx # Услуги
App.jsx # Основной компонент приложения
index.css # Глобальные стили

3. Запуск проекта 
cd <app>
npm start

4. Функционал компонентов
Header (Header.jsx)
Навигационное меню сайта
Логотип компании
Ссылки на Соц.сети

Hero (Hero.jsx)
Главный баннер
Призыв к действию

Services (Services.jsx)
Карточки услуг с изображениями

Projects (Projects.jsx)
Портфолио выполненных работ

Advantages (Advantages.jsx)
Секция преимуществ компании

ContactForm (ContactForm.jsx)
Валидация полей ввода

Footer (Footer.jsx)
Ссылки на социальные сети
Копирайт и юридическая информация

5. Стили и дизайн
CSS-модули: Каждый компонент имеет собственные стили
Адаптивный дизайн: Поддержка мобильных устройств
Современные UI-элементы: Интерактивные кнопки, формы, карточки
